A pre-configured starter project with the following features (all accurate at time of writing, very likely subject to change!);

- Webpack 4
- Babel 7
- Jest
- Enzyme
- ESLint & Prettier

## Main commands

```shell
# Run the dev server on port 8080
yarn start

# Generate a production build
yarn build

# Run ESLint
yarn lint

# Run unit tests
yarn test
```
